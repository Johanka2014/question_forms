Subject & Object Questions

Subject Questions

We use these when we want to ask about the subject of the sentence (the person or thing doing the action).

Statement: Somebody is knocking at the door.

Question: Who is knocking at the door?

Object Questions

We use these when we want to ask about the object of the sentence (the person or thing receiving the action).

Statement: Paul is reading something.

Question: What is Paul reading?

Forming Questions

Subject Question Rule

These are easy! The word order doesn't change. Just replace the subject with the question word (who, what, etc.).

Somebody won the match → Who won the match?

Object Question Rule

The word order changes. If there is an auxiliary verb (will, was, have, can), we put the subject after the auxiliary verb.

The structure is: Q-Word + Auxiliary + Subject + Verb

Allie will be at home → Where will Allie be?

If there is no auxiliary verb, we add one:

  • Present Simple: add do/does. (e.g., What does David eat?)
  • Past Simple: add did. (e.g., Where did they stay?)

If the verb has a preposition, it normally goes to the end.

Janet came with Michael → Who did Janet come with?
